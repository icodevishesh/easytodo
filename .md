create a minimal and modern todo app, with 2 sections open and completed. The todo will create tasks for current date. when the user create a new task it will show in open section and user can simply drag and drop the task from open to completed, marked as completed. The task has title, description (optional), createdAt(timestamp), dueDate(timestamp), completedAt(timestamp), id(uuid), edit and delete button. It also has a analytics page with weekly and monthly insights.

# Analysis of the Project Requirements

## Project Type

- **Desktop Application** built with **HTML, CSS, and JavaScript**.

## Core Features

1.  **Task Management**: Create, edit, delete, and update tasks.
2.  **Drag and Drop**: Move tasks between "Open" and "Completed" sections.
3.  **Analytics**: View weekly and monthly statistics.

## Data Model (per task)

- `id`: UUID
- `title`: String
- `description`: String (optional)
- `createdAt`: Timestamp
- `dueDate`: Timestamp
- `completedAt`: Timestamp
- `completed`: Boolean

## Technology Stack

- **Language**: react + typescript
- **Styling**: Tailwind CSS
- **Storage**: Dexie.js

- bg: #1c1c1c
